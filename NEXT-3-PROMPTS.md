```

```
Ты в сессии «MCP + НАВЫКИ» Тайга ИИ. Делаешь коннекторы и навыки production-grade.
Зона: server.py (MCP + skills), skills_run.py, skill_caps.py, taiga-web: mcp-panel / full-skills-panel.

P0-безопасность — MCP через SSRF-страж: проведи MCP-транспорт через уже существующий
