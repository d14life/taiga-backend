Как юзер по этому ходит — модель навигации

Структура экрана

  • Левый рельс — 7 иконок-групп, всегда видны. Наведение → название.
  • Клик группы → выезжает её панель только с нужными поверхностями (не 40 в кучу).
  • Чат — всегда в центре. Группы — слой поверх / док справа, не уводят с разговора.
  • Кастом-бар — закрепи 5-7 любимых поверхностей на главный экран (быстрый доступ).

Скорость для прокачанных

  • ⌘K командная палитра — печатаешь «озвучка» / «команда» / «дебаты» → прыжок к любой из 79 поверхностей мгновенно.
  • Слэш-команды в чате остаются (/image, /team …).
  • Поиск по всему: фичи, навыки, модели — один инпут.
  • Ничего не потеряно: каждая из 79 поверхностей нашла ровно одну полку.

Это карта-предложение (слой 1: информационная архитектура). Утвердишь группировку — я переношу навигацию на неё (левый рельс + ⌘K + группы), не ломая ни одной фичи; потом слой 2 — расхламление главного, слой 3 — премиум-эстетика. Скажи: «ок, строй» — или поправь группы (что переставить).
