## 1. Архитектура / 3-яруса + универсал-мозг (6)
- 3-яруса: Обычный (чат+код+без-цензуры) / Студия (картинки/видео/голос/3D) / Ultra (премиум-агент)
- УНИВЕРСАЛ-МОЗГ: любой чат может всё; мод = стиль/линза + дефолты, без жёстких лимитов
- Ultra = универсал-чат + ВСЕ табы + видимый пульт + ЭКСКЛЮЗИВ: только Ultra пересобирает систему через разговор
- Ultra-чат = разговорный дизайнер системы (генерит 3D-фон→ставит, меняет раскладку/тему/кнопки)
- 6 столов: Чат Дизайн Студия Агент Код Ultra

## 4. Конфиг-окна мульти-модельных режимов (9)
- **Конфиг-окно: lead-модель + сколько участников + какие подчинены `[GAP-FEATURE]`**

- Мозг: lead/driver + несколько craft/responder
- Мозг оркеструет N моделей (как агент), пресеты cheap/mid/top
- Различать Совет vs Ресёрч, Сравнить vs Ресёрч, Heavy vs Мозг

- Fusion-критик: кнопка проверки на деградацию/галлюцинацию
- Чат↔агент с памятью+тулзами; агент думает Мозгом/Советом
