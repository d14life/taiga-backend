- [ ] **Код** — UI готов · бэкенд ⏳

- [ ] **Мозг** — UI готов · бэкенд ⏳

  - для прода: Оркестратор lead-модели с function-calling, где «инструменты» — вызовы под-моделей; агрегация и финальный синтез на сервере.
- [ ] **Совет** — UI готов · бэкенд ⏳

- [ ] **Сравнить модели** — UI готов · бэкенд ⏳

- [ ] **Relay** — UI готов · бэкенд ⏳

- [ ] **Дебаты** — UI готов · бэкенд ⏳

- [ ] **Рерайт отказа** — UI готов · бэкенд ⏳

- [ ] **Ресёрч** — UI готов · бэкенд ⏳
