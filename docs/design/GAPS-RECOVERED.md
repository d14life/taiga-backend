## 🔴 P0 — критично (потеряно, Damir жал сильнее всего)
1. **Нативные reasoning-параметры** («changes the game»). Форвардить `reasoning_effort` / `thinking-budget` провайдерам, кто поддерживает; иначе фолбэк на промпт+токен-бюджет. **Усилие-слайдер должен реально углублять нативное мышление модели, а не только менять токен-кап/промпт.** Проверить per-provider.
2. **Баланс-aware cross-provider авто-роутинг.** При низком балансе/ошибке — авто-замена модели на рабочую КОПИЮ на профинансированном провайдере (у каждого провайдера копии моделей). Не просто стоп по бюджету — **реальный свитч маршрута.** (Повторял 6+ раз.)
3. **Баг tier-классификации.** `_tier` проверяет `'mini' in s` → ловит `miniMax` И `geMINI` → все MiniMax/Gemini ошибочно = small/fast → «в fast-режиме юзер берёт мощную модель, фрустрация». Фикс: word-boundary матч размера.

## 🟡 P1
4. **Temperature не форвардится** роутером на каждом пути (литеральный TODO в коде): сохраняется, но до модели не доходит. Дёшево, забыто.
5. **Утечки денег в метеринге.** Видео метрить по РЕАЛЬНОЙ цене (не ~256 текст-токенов); audio-генерация СПИСЫВАТЬ (сейчас проверяет баланс, но не дебетит ✗). Image метрит только текст-токены вместо реальной цены картинки.
6. **Юзер-ручка частоты консолидации памяти.** Сейчас фикс 6ч-idle; нет контроля, как часто память себя переписывает.
7. **Per-chat хуки — ПОТОК.** В окне настроек чата: создать хук (триггер + action-промпт) с кнопкой «ИИ поможет дописать промпт», полностью кастомно. Куски есть (`hooks-panel`/`lib/chat-hooks`), но ПОТОКА нет.
8. **Авто-уровень сложности.** Авто-режим выбирает глубину размышления ПО сложности вопроса (градация 1-5), не максит на простом. Сейчас только бинар `isTrivial→fastChatChain`.
9. **Token-cap per-model для reasoning.** Хард-кап 16384 режет нативное мышление пополам. Не обрезать мышление — тюнить кап по reasoning-overhead модели; лучше spend-cap+уведомление+дешёвая альтернатива, чем обрезка. Damir: «это базовое».
10. **Гибрид-RAG.** BM25 + RRF reranker + pgvector + HNSW-индекс + предфильтр `user_id`. Brute-force не эффективен на мультиюзере. Damir: «самый большой скачок точности».
