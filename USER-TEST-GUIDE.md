# Тайга — гайд юзер-тестов (что, где, как, ожидаемо)

Все фичи этой серии сессий + где они в UI + как проверить. Перед тестами: backend :8777 и frontend
:3000 должны отдавать 200. Открой приложение → ниже по областям.

---

## A. Строка ввода (композер) — быстрые тумблеры/кнопки
| Фича | Где | Как тест | Ожидаемо |
|---|---|---|---|
| **Подсказки «что дальше»** | под ответом ИИ, над вводом | задай содержательный вопрос (код/тема) | 2-4 кнопки следующего шага; тап = отправить. На «ок»/«привет» кнопок НЕТ (гейт) |
| **Пресет-чипы** | ряд над вводом «пресеты» | тап по пресету (Юрист/Кодер/Аналитик…) | грузится ВЕСЬ комплект: режим+модель+персона+флаги |
| **Авто-режим** (тумблер) | у поля ввода | выключи → задай трудный вопрос | отвечает ОДНА модель, без тихого Мозга (gap 5) |
| **Анти-сползание** (тумблер) | у поля ввода | вкл → длинный чат с заданной ролью | роль не уплывает (напоминание перед последним сообщением) |
| **Характер** (кнопка) | у поля ввода | тап → выбери «Критик»/«Архитектор»/… | чат отвечает как персона; Архитектор авто-вкл держатель |
| **Глубина** (слайдер) | у поля ввода | подвинь 1-5 | ниже=быстро, выше=reasoning_effort больше |
| **Модель** (пикер) | у поля ввода | выбери явную модель | именно она отвечает (не свапается даже при тесном балансе) |

## B. Режимы ответа
| Фича | Где | Как тест | Ожидаемо |
|---|---|---|---|
| **Мозг** | режим-пилюля | трудный вопрос | дешёвый ведущий → сильный эксперт; помнит тебя + тулзы |
| **Совет** | режим-пилюля | выбери 2-5 моделей | веер мнений → синтез |
| **Глубоко/Ультра** + **Источник правды** | пилюли/тумблер | вкл источник правды + ультра | многошаговый ответ с цитатами [N] |
| **Бейдж уверенности** ⭐ | под ответом (вкл «авто-проверку») | включи авто-проверку → задай вопрос | бейдж «уверен 92%»/«скорее догадка 40%»; на ФАКТ-ошибке (напр. «Сидней=столица Австралии») → предупреждение «возможна галлюцинация» |

## C. Картинки
| Фича | Где | Как тест | Ожидаемо |
|---|---|---|---|
| **Картинка в чате** | обычный чат | «нарисуй кота» | модель зовёт generate_image → картинка в ленте |
| **Студия** | вкладка медиа | вариации/правка/апскейл/видео/сцены | про-верстак (то, чего нет в быстрой картинке чата) |

## D. Агент-таб («Агент»)
| Фича | Где | Как тест | Ожидаемо |
|---|---|---|---|
| **Обзор** (дашборд) | Агент → Обзор (дефолт) | открой | KPI (всего/успех%/средняя/за-24ч) + графики + лента прогонов |
| **Готовые пайплайны** | Агент → Режимы → внизу | жми «🔬 Глубокий ресёрч» → впиши тему | результат течёт по шагам в чат |
| **⚖️ Дебат Архитектор↔Критик** | Агент → Режимы → карточка | впиши мутную идею («синхронизация файлов») | ping-pong раундов; на развилке — ОКНО-ВОПРОС с кнопками-вариантами (B1) + textarea; в конце чертёж |
| **Команда** («Сделать агентом») | над лентой / Режимы | цель → собери команду | шаги: Цель/Команда/Бюджет/**Мышление**(норм/Мозг/Совет)/**Движок+режим**(модель·парал/послед·**Глубина** лёгкий/полный/авто) |
| **Источник правды** | Агент → под-таб | тумблер grounded + **«Экспорт памяти»** | заземление; экспорт качает JSON всей памяти |
| **Песочница/Инструменты/Оценки/Ralph/Прогоны/Задачи/Автоматизации** | Агент → под-табы | открой каждый | панели на месте |

## E. Под капотом (прозрачно — проверяется поведением)
- **Агент-воркеры = полные чаты** (память+тулзы): дай агенту факт из прошлого + «посчитай X» → вспомнит+посчитает.
- **Автономия воркеров**: на мутной задаче агент ДЕЛАЕТ (помечает «допускаю»), не переспрашивает.
- **Гибрид-RAG** (дефолт): загрузи док, спроси по терминам/числам → точнее (dense+keyword+RRF).
- **Авто-здоровье моделей**: мёртвую модель система обходит сама, через ~30 мин пробует снова.
- **Лестница бюджета**: тесный баланс → ↓глубину (твоя модель цела); свап только в авто-режиме.
- **WAL-чекпойнт**: раз в 30 мин (БД не пухнет).

## ⚠️ Чтобы тестеры УВИДЕЛИ всё, включи заранее:
- **«Авто-проверку»** (иначе бейдж уверенности не покажется — дефолт OFF, это один вызов на ответ).
- **«Источник правды»** если тестишь цитаты/grounding.
- Загрузи 1-2 документа (для теста RAG/гибрида).

## Известные шероховатости (не баги, фон):
- Бейдж уверенности честен ровно настолько, насколько умён судья-верификатор (может пропустить редкий факт).
- Живой документ (анти-сползание) наполняется в основном из дебата «Взять как чертёж».
- Композер плотный — много кнопок; при желании свернём редкие в «⚙ ещё».
